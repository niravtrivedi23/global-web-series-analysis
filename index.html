🌍 Global Web Series Analysis Dashboard (2000–2024)

This project analyzes 3,400+ web series released between 2000–2024 across 42 countries and major OTT platforms like Netflix, Prime Video, Disney+, and more. Built with Python for data preparation and an interactive Power BI dashboard for visual storytelling, it uncovers streaming-platform trends, global release patterns, language & genre insights, top-rated shows, and audience-vote behavior.
